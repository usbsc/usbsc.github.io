### 🎵 [Frequency Tool](https://usbsc.github.io/frequency-tool/)
A real-time audio frequency analyzer built with Web Audio API and HTML5 Canvas.

**Features:**
- Real-time microphone audio capture
- FFT frequency spectrum analysis
- Interactive visualizations (spectrum, waveform, power, waterfall)
- Adjustable frequency range and controls
- Mobile responsive design

**Links:**
- 🌐 [Live Demo](https://usbsc.github.io/frequency-tool/)
- 📝 [Source Code](https://github.com/usbsc/frequency-tool)
